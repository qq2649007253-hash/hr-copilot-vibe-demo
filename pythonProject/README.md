🤖 HR-Copilot: 基于大模型的智能简历筛选与分析助手

项目定位：一款为 HR 和面试官打造的 AI 辅助工具，通过大语言模型自动解析简历、匹配 JD（岗位描述）并生成面试建议，解决海量简历初筛的效率瓶颈。

开发模式：Vibe Coding (自然语言驱动全栈开发)

💻 本地运行指南

如果你希望在本地开发环境中运行源码：

# 1. 克隆仓库
git clone [https://github.com/opn48/hr-copilot.git](https://github.com/opn48/hr-copilot.git)

# 2. 安装依赖
pip install streamlit

# 3. 运行应用
streamlit run app.py


📂 项目结构

app.py: Streamlit 前端与核心交互逻辑

REQUIREMENTS.md: 特定领域的痛点分析与用户研究报告
